Keyboard Shortcuts & Help

Drawing & Editing

D - Switch to Draw mode

E - Switch to Erase mode

Click & Drag - Draw a new rectangle

Shift + Click - Start drawing on occupied cells

Click - Select an element

Alt + Click - Cycle through overlapped elements

Ctrl/Cmd + Click - Multi-select elements

Esc - Cancel drawing / Clear selection

Navigation

Mouse Wheel - Zoom in/out (pointer-centered)

Middle Mouse Drag - Pan viewport

Space + Left Drag - Pan viewport (alternative)

F - Focus selected element (center in view)

Selection Editing

Arrow Keys - Nudge selected element (1 cell)

Shift + Arrows - Nudge selected element (5 cells)

Alt + Arrows - Resize selected element (1 cell)

Alt + Shift + Arrows - Resize selected element (5 cells)

Delete / Backspace - Delete selected element

Undo & Redo

Ctrl/Cmd + Z - Undo last action

Ctrl + Y or Ctrl/Cmd + Shift + Z - Redo

Help

H or ? - Show this help dialog
